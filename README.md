#  Library Management System (Java Console Application)

##  Overview
This is a simple **Library Management System** built using Java.  
It is a console-based application that allows users to manage books, student memberships, and borrowing records.

The project demonstrates core **Data Structures and Algorithms (DSA)** concepts like:
- Arrays
- Searching
- Sorting (Bubble Sort)
- Stack
- Linked List
- Hashing

---

##  Features

### 📖 Book Management
- Add book details (ID, Title, Author, Category)
- Display all books
- Search books by title
- Sort books by ID

### 👨‍🎓 Student Membership
- Add student (Stack - Push)
- Remove student (Stack - Pop)
- Display student list

### 📕 Borrow Records
- Maintain student borrow records using Linked List
- Add new borrowing data
- Display all borrowing records

### 🔢 Book ID Allotment
- Hashing technique for assigning book IDs

### 📂 File Handling
- Stores book records in a text file  
  (See: `Books Record.txt`) :contentReference[oaicite:0]{index=0}
